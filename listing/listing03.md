Чем отличаются RWMutex от Mutex?

Ответ:
```
Mutex представляет собой стандартный механизм блокировки в Go, который позволяет только одной горутине захватить
блокировку в определенный момент времени.

RWMutex представляет собой расширение Mutex, которое поддерживает два типа блокировок: блокировку для чтения и
блокировку для записи. Блокировка для чтения позволяет нескольким горутинам получать доступ к общему ресурсу
для чтения одновременно, при условии, что ни одна горутина не пытается получить доступ на запись.

```
