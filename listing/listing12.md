Что выведет данная программа и почему?

```
func main() {
  n := 0
  if true {
     n := 1
     n++
  }
  fmt.Println(n)
}
```

Ответ:
```
0

Это происходит из-за того, что внутри блока if true определена новая переменная n, которая имеет область видимости
только внутри этого блока (то есть, она локальная). Эта новая переменная n закрывает доступ к переменной n,
объявленной перед блоком if. И тем самым, когда происходит n++, увеличивается значение локальной переменной, а не
глобальной. Поэтому, вывод будет также 0.

```
