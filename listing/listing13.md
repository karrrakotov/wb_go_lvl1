Что выведет данная программа и почему?

```
func someAction(v []int8, b int8) {
  v[0] = 100
  v = append(v, b)
}

func main() {
  var a = []int8{1, 2, 3, 4, 5}
  someAction(a, 6)
  fmt.Println(a)
}
```

Ответ:
```
[100 2 3 4 5]

Первое значение слайса изменится с 1 на 100 (v[0] = 100), произойдет это, потому что, когда слайс передается в параметр функции (он передается в виде адреса) и любые манипуляции с уже существующими значениями, будут изменять непосредственно сам слайс. А вот 6-ой (новый) элемент не добавиться. Так как append создает непосредственно новый слайс в памяти, а не тот же самый, а так как, мы его не возвращаем и не присваиваем, новое значение мы не увидим.
```
