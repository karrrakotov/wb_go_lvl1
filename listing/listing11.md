Что выведет данная программа и почему?

```
func main() {
  wg := sync.WaitGroup{}
  for i := 0; i < 5; i++ {
     wg.Add(1)
     go func(wg sync.WaitGroup, i int) {
        fmt.Println(i)
        wg.Done()
     }(wg, i)
  }
  wg.Wait()
  fmt.Println("exit")
}
```

Ответ:
```
Непредсказуемый
Возможен deadlock

Код программы выше неверный. 
sync.WaitGroup не должен передаваться по значению в анонимную функцию, потому что, таким образом, каждая горутина будет работать с копией своего собственного WaitGroup. В таком коде нет никакой синхронизации.
```
