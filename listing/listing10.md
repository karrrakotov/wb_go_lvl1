Что выведет данная программа и почему?

```
func update(p *int) {
  b := 2
  p = &b
}

func main() {
  var (
     a = 1
     p = &a
  )
  fmt.Println(*p)
  update(p)
  fmt.Println(*p)
}
```

Ответ:
```
1
1

В данном случае, в функции update, мы передаем указатель на int и пытаемся изменить его значение на адрес переменной b.
Однако в Go, аргументы передаются по значению, а не по ссылке. Это означает, что при передаче указателя p в функцию update,
создается копия указателя p, и все изменения, произведенные на копии, не будут влиять на исходный указатель p в функции main.

```
