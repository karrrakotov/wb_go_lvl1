Что выведет данная программа и почему?

```
func main() {
  slice := []string{"a", "a"}

  func(slice []string) {
     slice = append(slice, "a")
     slice[0] = "b"
     slice[1] = "b"
     fmt.Print(slice)
  }(slice)
  fmt.Print(slice)
}
```

Ответ:
```
[b b a][a a]

[b b a] - печатается с анонимной функции. Что происходит? Мы передаем слайс в анонимную функцию, затем делаем append, когда мы это делаем, в памяти создается новая копия нашего слайса и у этой копии, мы уже меняем значения  slice[0] = "b" и  slice[1] = "b". 

[a a] - а ниже, за пределами анонимной функции, мы печатаем наш изначальный слайс, который никак не был изменен.

```
